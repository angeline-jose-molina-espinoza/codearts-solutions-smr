# CodeArts Solutions SMR

Este repositorio contiene las prácticas realizadas durante el ciclo de Sistemas Microinformáticos y Redes (SMR).

Incluye configuraciones de servidores, máquinas virtuales, redes y pruebas realizadas en entornos empresariales.

Se ha implementado el uso de ramas en Git:
- main: rama principal estable
- prod: rama secundaria para pruebas independientes
